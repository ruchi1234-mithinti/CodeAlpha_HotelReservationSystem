# Hotel Reservation System

A simple Java console-based hotel reservation system developed using OOP concepts.

## Features
- View available hotel rooms
- Room categorization (Standard, Deluxe, Suite)
- Book hotel rooms
- Cancel reservations
- Payment simulation
- Room availability tracking
- Menu-driven console interface
- Exception handling for invalid inputs

## Technologies Used
- Java
- OOP Concepts
- ArrayList

## Project Structure
- Main.java
- Room.java
- Hotel.java
- Reservation.java

## How to Run
1. Open the project in IntelliJ IDEA
2. Run Main.java
3. Use the menu options in the console
